# Проект #2: Анализ текстовых данных на Flask

Проект посвящен созданию простейшего веб-приложения на микрофреймворке `Flask`.

## Сценарий использования

| Номер | Действие | Результат | Балл |
|---|---|---|---|
| 1 | Пользователь переходит по корневому адресу | Отображается страница с формой для выбора файла | 1 |
| 2 | Пользователь выбирает файл для обработки | Проверка корректности файла (на клиенте) |
| 2-A | ...выбранный файл - корректный | Отображение элементов интерфейса для выбора действия | 1 |
| 2-B | ...выбранный файл - некорректный | Сообщение-подсказка об ошибке, возврат к пункту 2 | 1 |
| 3 | Пользователь выбирает желаемое действие и нажимает кнопку | Отправка формы, обработка формы | 1 | 
| 3-A | ...действие - "Статистика" | Перенаправление на страницу `/stats`, отображение статистики в табличном виде | 1 |
| 3-B | ...действие - "Топ-слова" | Перенаправление на страницу `/top`, отображение топ-слов в табличном виде и изображения с облаком слов | 3 |
| 3-D | ...не удалось обработать файл | Сообщение-подсказка об ошибке, возврат к пункту 2 | 1 |
